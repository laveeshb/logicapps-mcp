# Operations

Step-by-step guides for Logic Apps operations. Each file covers a complete operation including prerequisites, tools, steps, and post-operation tasks.

## Available Operations

| Operation | Description |
|-----------|-------------|
| [Clone](clone.md) | Clone a Consumption Logic App to a Standard Logic App |

## Structure

Each operation file follows a consistent format:

1. **Overview** - What the operation does
2. **Prerequisites** - What must exist before starting
3. **Tools** - MCP tools used in this operation
4. **Steps** - Detailed step-by-step workflow
5. **Post-Operation** - Tasks to complete after the operation
6. **Common Issues** - Troubleshooting and known limitations
